Help
Install Android
  1. Wejdź w powyższy link przez przeglądarkę (preferencyjnie Chrome)
  2. U dołu ekranu powinno pojawić się wyskakujące powiadomienie o treści zbliżonej do "Dodaj Śpiewnigator do ekranu głównego"
  3. Przekilkaj dalej jeśli potrzeba
 Install IOS
  1. Wejdź w powyższy link przez Safari (koniecznie!)
  2. Wejdź w udostęnianie (share) -> dodaj do ekranu głównego (add to home screen)
  3. Przeklikaj dalej jeśli potrzeba
